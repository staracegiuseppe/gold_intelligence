?Guida: Gmail con App Password

    

      
Come configurare Gmail:

      
1. Vai su myaccount.google.com → Sicurezza

      
2. Abilita Verifica in 2 passaggi

      
3. Cerca Password per le app → crea password per "Mail"

      
4. Usa la password generata (16 caratteri) come smtp_password

      

        smtp_host: smtp.gmail.com

        smtp_port: 587

        smtp_user: tua@gmail.com

        smtp_password: xxxx xxxx xxxx xxxx

        smtp_tls: true
      

      
Per altri provider:

      

        Outlook: smtp-mail.outlook.com:587

        Yahoo:   smtp.mail.yahoo.com:587

        iCloud:  smtp.mail.me.com:587
